## Structure

This repository is structured as follows:

```
packages
├── core
├── react
└── uvslides

site
├── public
└── src
    ├── app
    │   ├── (common)
    │   └── docs
    ├── components
    │   └── ui
    ├── lib
    └── mdx
```

| Path                           | Description                                   |
| ------------------------------ | --------------------------------------------- |
| `core`                         | The package of framework agnostic components. |
| `react`                        | The React components of the package.          |
| `uvslides`                     | The React components of the package.          |
| `site`                         | The Next.js application for `uvcanvas.com`.   |
| `site/src/app/docs`            | The MDX docs for the package.                 |

## Development

### Fork this repo

You can fork this repo by clicking the fork button in the top right corner of this page.

### Clone on your local machine

```bash
git clone https://github.com/your-username/uvcanvas.git
```

### Navigate to project directory

```bash
cd pacakges/xxx
```

### Create a new Branch

```bash
git checkout -b my-new-branch
```

### Install dependencies

```bash
yarn
```

### Run development server

```bash
yarn dev
```

This will watch all the packages and start a docs development server to support hot updates to all the code.
